# Nest-Fitness-Server

- Stalled version with Go [Gin Gonic version](https://github.com/FACorreiaa/fitness-server-app)
- Older version using express mockup [Express version](https://github.com/FACorreiaa/node-fitness-app-server)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
